JustWave Project: Downloads

You can download sources at GitHub or use one of the following buttons.

Production release

justwave.zip
justwave.tgz

Files for production release are:

  1. JustWave.class.php - PHP class for creating png waveforms of audio using FFMPEG when needed
  2. justwave.ajax.php - helper server script for joining JustWave.class.php with client-side scripts (e.g. justwave.player.js)
  3. justwave.player-min.js - client-side script (minified version) for creating waveform-like players on audio elements in HTML page
  4. justwave.player.css - CSS file for styling players created by justwave.player.js
